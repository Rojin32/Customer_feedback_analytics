# Customer Feedback Analytics System
## Features
- Customer Feedback Collection
- Sentiment Analysis using TextBlob
- SQLite Database
- Analytics Dashboard
- Sentiment Distribution
- Rating Analysis
- Product Comparison
- Keyword Analysis
## Technologies
- Python
- Flask
- SQLite
- Streamlit
- Pandas
- Matplotlib
- TextBlob
## Run Flask App
python app.py
## Run Dashboard

python -m streamlit run dashboard.py
